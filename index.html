Part B One scatter plot and two datasets

The plot

The scatter plot contains data about different crimes across different districts in San Fransisco for 2003 and 2015. The x-axis corresponds to the cases of prostitution and the y-axis the cases of car thefts. The radius of the circles are probational with the total number of crimes within the district.

The axes are the same for both plots in order to make it easier to compare them and spot trends, and the plot is fully responsive and will adjust if the window is resized.

Hover the mouse above the circles to get the exact numbers.

The data

The plots shows that especially Northern, Tenderloin and Mission have reduced the two kinds of crimes from 2003 to 2015. These were also the three districts with by far most prostitution cases. Southern and Taravan are the only districts that have an increase in the number of prostitution.

The total number of crimes per district doesn't change much over the years.

Part D Visualizing geodata

The plot

The plot shows a k-means clustering of prostitution in San Francisco in the period 2003-2015. Each district is color coded similar to previous plots and the dots are cases of prostitution. The squares are the centroids for dots of same color.

The plot are fully responsive and the buttons can be hovered to get a preview of the data of that view.

The data

To make the web page load faster we took some actions make the data files as small as possible, however, we trained the model on all the data before any modifications. To reduce the data we removed all duplicate points. We created the json file crimes.json containing first the positions of the centroids for all k-tests and then the data points with an associated index to the centroid of each k-test. With these modifications we managed to get data file size of 116kb. The Python Notebook can be found here: Link
