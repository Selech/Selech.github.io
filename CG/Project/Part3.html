Project part 3

Controls

The camera can be controlled freely using the keyboard. It is also possible to use the buttons to snap to predefined views.

Moving the camera:
Use ← and → to move along the x-axis
Use ↑ and ↓ to move along the y-axis
Use + and - to move along the z-axis

Rotating the camera:
Use A and D to rotate around the y-axis
Use W and S to rotate around the x-axis

Description

In this scene we have implemented parallax correction which will result in more realistic reflections. This is specially visible in the reflections of the shadows because the ground is close to the mirror ball. Compared to project part 2 this solution has almost the same performance.
However, our implemenetation is not perfect and minor visual artifacts can sometimes be found, especially in the inter-reflections. That said, we belive it is a good solution and it shows that parallax correction gives better results with almost no performance loss.
